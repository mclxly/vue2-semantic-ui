## build 后的文件尺寸

* client.js: 891kb
* styles.css: 532b

To start:

```bash
$ npm install
```

To develop:

```bash
$ npm run dev```

To build for production:

```bash
$ npm run build
```

To lint you code:

```bash
$ npm run lint
```

测试编译后的静态文件:

```bash
$ cd dist
$ python -m SimpleHTTPServer 8000
```
